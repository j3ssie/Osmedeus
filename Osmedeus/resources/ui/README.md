UI
===========
Web UI for Osmedeus based on my [Jaeles UI](https://github.com/jaeles-project/ui/).

## Installation

### [React source](https://github.com/j3ssie/UI/tree/master/Osmedeus-ui)

```
yarn install
npm start
```
